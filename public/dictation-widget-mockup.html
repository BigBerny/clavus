Clavus Dictation Widget — Feature Spec

Overview

A floating pill widget that appears when the user triggers dictation via a global keyboard shortcut. Records speech, transcribes it, and immediately inserts text into the active app on release. No confirmation step.

Widget Layout (left to right)

  • Lock/Stop — Glass circle. Lock icon while holding shortcut (click to go hands-free). Becomes stop icon after locking.
  • Recording indicator — Pulsing dot, animated waveform, timer.
  • Format group — A single row of selectable options for how the text should be formatted/rewritten before insertion. Order:
    • App name (text, no icon) — Only shown for generic apps (browsers, editors). Shows the app name or field type (e.g. "Arc", "VS Code", "URL"). Not shown when the foreground app matches a known format (Slack, Mail).
    • Insert As (paste icon) — Inserts raw transcription, no rewriting. Always second (or first when no app name).
    • Slack (icon) — Rewrites as casual Slack message.
    • Messaging (icon) — Rewrites as short casual message (iMessage, WhatsApp, Telegram style).
    • Email (icon) — Rewrites with greeting/sign-off.
    • Prompt (icon) — Rewrites as structured AI prompt.
  • Translate toggle ("EN", text) — Independent toggle that can be combined with any format. When active, translates the text to English before inserting. Works as a modifier, not a format — e.g. Slack + EN = Slack-formatted message translated to English.
  • Start Chat (icon, indigo tint) — Opens Clavus, creates new thread, sends transcript as first message.

Auto-Selection Rules

  • If the foreground app is Slack: the Slack icon is pre-selected. No app name shown.
  • If the foreground app is Mail: the Email icon is pre-selected. No app name shown.
  • If the foreground app is a browser or editor: the app name (e.g. "Arc") appears as text and is selected by default. If the cursor is in a specific field type (URL, search), that type replaces the app name (e.g. "URL").
  • User can override by clicking any other format. The previous selection deselects.

Interaction Flow

1.User presses and holds global shortcut. Widget appears. Recording starts.
2.Waveform animates with voice, flattens when silent. Live transcript above pill.
3.User can click a format or Start Chat while recording.
4.User can click Lock to go hands-free. Lock becomes Stop.
5.Releasing shortcut or clicking Stop immediately inserts text (reformatted if a format was selected). Widget disappears.
6.Clicking Start Chat ends recording, opens Clavus, sends transcript as first message.

Visual Design

Follows the Clavus iOS 26 liquid glass system: warm oklch backgrounds (hue 50), blur(20px) saturate(200%), inner glow, soft borders. Format icons are icon-only circles; the app name is a text-only pill. Start Chat has an indigo glass tint.
