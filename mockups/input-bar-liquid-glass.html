Recommended directionOption A keeps transient generation/queue/transcription state in a distinct top control layer, which matches the Liquid Glass principle best and preserves the mic action.
Implementation noteOn the web this is an approximation: backdrop blur, edge highlights, saturation, and a solid fallback for reduced transparency.
Behavior ruleThe composer action row should never replace the dictation button just because a response is generating. Stop is a sibling control, not a mode replacement.
